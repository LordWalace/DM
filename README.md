# DM – DayMind

DM (DayMind) é um aplicativo multiplataforma para organização de tarefas diárias com suporte a Inteligência Artificial.

O usuário pode criar atividades manualmente ou escrever textos naturais que são interpretados pela IA e transformados em tarefas com data, horário e notificações automáticas.

## Funcionalidades
- Criação manual de tarefas
- Criação automática usando IA
- Notificações sincronizadas
- Calendário integrado
- Modo offline
- Sincronização em nuvem
- Histórico de produtividade

## Tecnologias
- Frontend Desktop: Electron + React + TypeScript
- Mobile: React Native (Expo)
- Backend: NestJS (Node.js + TypeScript)
- Banco de Dados: PostgreSQL
